# Personal Finance Manager

A Python-based console application to track personal expenses.

## Features
- Add daily expenses
- View all expenses
- View total expenses
- Category-wise expense summary
- Monthly expense summary
- Export category summary to CSV

## Technologies Used
- Python
- CSV file handling

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:
   python main.py
